# Tomer Hershkovitz Portfolio

Static portfolio site built for GitHub Pages.

## Files

- `index.html` - page structure and content
- `styles.css` - visual design and responsive layout
- `script.js` - small interaction layer for scroll reveals and footer year

## Publish With GitHub Pages

1. Create a GitHub repository and push this folder to `main`.
2. Open the repository on GitHub.
3. Go to `Settings` -> `Pages`.
4. Under `Build and deployment`, choose `Deploy from a branch`.
5. Select branch `main` and folder `/root`.
6. Save, wait for the deployment to finish, and open the generated URL.

If the repository name is `yourusername.github.io`, the site will be served from the root domain. Otherwise GitHub Pages will publish it under `https://yourusername.github.io/repository-name/`.

## Quick Local Preview

Run:

```bash
python3 -m http.server 8000
```

Then open `http://localhost:8000`.
